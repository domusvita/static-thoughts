# static-thoughts

A minimal static site hosted on GitHub Pages.

## Structure

| File | Description |
|------|-------------|
| `index.html` | Main page |
| `styles.css` | Stylesheet |

## Deployment

The site deploys automatically to GitHub Pages when changes are pushed to the `main` branch via the workflow in `.github/workflows/deploy.yml`.

Enable GitHub Pages in the repository settings and set the source to **GitHub Actions**.
